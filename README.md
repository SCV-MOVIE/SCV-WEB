## How To

### Environment

```bash
OS: macOS 13.1 (Ventura)
```

```env
# .env

NEXT_PUBLIC_API=[API ENDPOINT]
```

### Install

```bash
yarn install # or yarn
```

### Run

```bash
# in Dev.
yarn dev

# in Prod.
yarn build
yarn start
```
